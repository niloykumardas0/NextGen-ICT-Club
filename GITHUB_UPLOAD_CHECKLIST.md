# GitHub Upload Checklist for NextGen ICT Club

GitHub-এ আপনার প্রোজেক্ট আপলোড করার সময় নিচের তালিকাটি অনুসরণ করুন। এটি নিশ্চিত করবে যে আপনার সাইটটি Netlify-এ সঠিকভাবে কাজ করছে।

### ✅ এই ফাইল এবং ফোল্ডারগুলো অবশ্যই আপলোড করবেন:
1.  **src/** (আপনার সব সোর্স কোড এখানে আছে)
2.  **public/** (ইমেজ এবং অন্যান্য পাবলিক ফাইল)
3.  **docs/** (ডেটাবেজ ব্লুপ্রিন্ট/backend.json)
4.  **package.json** (ডিপেন্ডেন্সি লিস্ট)
5.  **package-lock.json** (ভার্সন কন্ট্রোল)
6.  **next.config.ts** (Next.js কনফিগারেশন)
7.  **tailwind.config.ts** (Tailwind কনফিগারেশন)
8.  **tsconfig.json** (TypeScript কনফিগারেশন)
9.  **components.json** (ShadCN কনফিগারেশন)
10. **.gitignore** (অপ্রয়োজনীয় ফাইল বাদ দেওয়ার জন্য)
11. **README.md** (ইন্সট্রাকশন ফাইল)

### ❌ এই ফাইল এবং ফোল্ডারগুলো কখনোই আপলোড করবেন না:
1.  **node_modules/** (এটি অনেক বড় এবং Netlify নিজে থেকেই এটি তৈরি করে নেয়)
2.  **.next/** (এটি বিল্ড ফোল্ডার, যা আপলোড করার প্রয়োজন নেই)
3.  **.env** (এখানে আপনার সিক্রেট API Key থাকে, এটি আপলোড করলে হ্যাক হওয়ার ভয় থাকে)
4.  **.DS_Store** (ম্যাক ইউজারদের জন্য অটো-জেনারেটেড ফাইল)

---

### 🚀 আপলোড করার পর Netlify-তে যা করবেন:
GitHub-এ আপলোড শেষ হলে Netlify-তে গিয়ে **Environment Variables** সেকশনে আপনার Firebase-এর API Key গুলো বসিয়ে দিবেন। key গুলোর নাম `README.md` ফাইলে দেওয়া আছে।
