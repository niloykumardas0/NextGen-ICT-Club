# **App Name**: Vanguard Olympiad

## Core Features:

- Smart Proctored Exam Tool: A browser-locked, timed examination interface that uses an AI tool to monitor focus and prevent navigation away from the exam tab.
- Olympiad Command Center: A centralized dashboard for participants to manage their exam registration, track preparation status, and download admit cards.
- Global Tech Leaderboard: Real-time, Firestore-synced rankings with automatic scoring based on examination time and accuracy.
- Club Hub Portal: Registration and profile management for club members, featuring dynamic digital membership cards with unique QR identifiers.
- Ambassador Growth Tracker: A referral-linked portal for Campus Ambassadors to track recruitments and performance statistics through custom codes.
- Automated Certification Engine: Instant generation and archival of PDF merit certificates accessible through the user profile upon result publication.
- Unified Admin Dashboard: An administrative oversight suite for managing participant records, questions, notices, and club membership applications.

## Style Guidelines:

- Primary Color: Cyber Violet (#7C3AED) evoking high-energy innovation and futuristic technology.
- Background Color: Deep Void Grey (#0D0D12), providing a dark, immersive tech-inspired canvas with 15% saturation.
- Accent Color: Electric Cyan (#0EA5E9) for critical action points, navigation highlights, and neon glow effects.
- Headline and body font pairing: 'Space Grotesk' for tech-centric headers and 'Inter' for highly legible, objective body text.
- Glassmorphism architecture using semi-transparent containers, backdrop filters (blur), and 1px borders with subtle gradients.
- Monolinear, neon-stroked icons that represent digital connectivity, artificial intelligence, and academic competition.
- Sleek CSS-driven micro-interactions, such as glowing button hover states and smooth sectional transitions for a high-end feel.
